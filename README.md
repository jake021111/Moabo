# MOABO
05.14
프로젝트 개요: 키워드를 통해 여러 미디어의 알고리즘을 모아 한번에 볼수 있게 추천해주는 서비스(모아보)

개발동기: 바이브 코딩으로 소프트웨어를 만들라는 과제를 통해 내가 무엇을 필요로 할까 생각해 본 결과
평소에 여러 ott, 게임, 등을 하거나 볼때마다 각각의 앱에 들어가서 찾기 내 취향에 맞는 여러 작품들을 찾기 귀찮았는데 
하나의 사이트에서 다 해결 할 수 있다면 내가 쓰기에도 편할 것 같아서 제작하자고 합니다.



주요 기능

키워드 다중 선택 — 4개 카테고리에 공통 적용되는 12개 키워드(판타지·모험·로맨스·액션·SF·일상·코미디·추리·힐링·스릴러·한국작품·성장) 중 여러 개 선택
카테고리별 추천 — 선택 키워드에 매칭되는 작품을 게임·애니·영화·웹툰 각 3개씩 카드로 표시
새로고침 추천 — 같은 키워드 조건에서 다른 작품으로 재추천
북마크 — 작품을 저장하고 별도 화면에서 모아보기 (localStorage로 영속 저장)
키워드 수정 — 이전 선택을 유지한 채 키워드 선택 화면으로 복귀

기술 스택

프레임워크: React + Vite
언어: JavaScript (ES6+)
스타일: CSS (모바일 반응형)
데이터: src/data/works.json (작품 60개 / 키워드 12개 / 카테고리 4개)
저장소: 브라우저 localStorage (북마크 영속 저장)
